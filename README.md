# Cal.com Clone

A simplified clone of Cal.com built with:
- **Frontend**: React (Vite)
- **Backend**: Node.js (Express)
- **Database**: MySQL

## Features
- Event Type Management
- Availability Scheduling (with Timezone support)
- Public Booking Page
- Dashboard & Bookings Management
- Dark Mode
- Responsive Design

## Setup
1. `cd server` && `npm install`
2. `cd client` && `npm install`
3. Setup MySQL database using `server/schema.sql`
4. Configure `.env` files
5. Run `npm run dev` in both folders.
